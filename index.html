Gestor de TI na Ginco Urbanismo, em Cuiabá — onde Lidero uma equipe de Tecnologia, com atuação em infraestrutura, Hyper-V, segurança, helpdesk e o desenvolvimento de ferramentas internas. Minha rotina mistura administração / Infra / Liderança e aos poucos implementando a escrita de código para automação de processos.

Desenvolvi alguns sistemas completos do zero: o Ginco Flow, plataforma de workflows com integração ao ERP UAU; o App Financeiro, uma solução full-stack de gestão pessoal; e o Painel TV, dashboard em tempo real para exibição corporativa.

Minha trajetória caminha cada vez mais em direção ao desenvolvimento de software — onde a profundidade técnica de infra encontra a expressividade do código.

02 projetos

01
Ginco Flow
Plataforma interna de workflows para substituir o Fluig. Solicitações de pagamento com preenchimento automático via ERP UAU, aprovação multi-etapas, notificações em HTML e módulos de férias e distrato.
Node.js React MySQL JWT Nodemailer mssql
02
App Financeiro
Aplicação full-stack de gestão financeira pessoal. Controle de lançamentos, parcelamentos inteligentes, gráficos e acesso mobile via Cloudflare Tunnel. Frontend no Netlify, backend local com XAMPP.
Node.js Express MySQL Vanilla JS Netlify Cloudflare
03
Painel TV
Dashboard informativo em tempo real para exibição em TV corporativa. Projeto de portfólio desenvolvido sob a marca Beta Soluções, com acesso remoto e suporte a PWA mobile.
Node.js Express HTML/CSS Cloudflare PWA
